Systèmes Véhicule - Description Fonctionnelle

1. Systèmes d'éclairage (Lighting System I et II)

Le système d'éclairage est géré principalement par le Calculateur de réseau de bord (J519) et l'électronique de colonne de direction (J527).

- Fonction :

Commande des feux de position, de croisement, de route et des clignotants.

- Messages CAN :

  • 2C1h : Envoyé par le J527 toutes les 100 ms pour indiquer l'état des commutateurs de phares/clignotants.
  • 531h : Etat du J519 envoyé toutes les 50 ms.
  • 631h : Message d'état secondaire envoyé par le J519 (intervalle d'1 s).

- Interaction ECU :

Le commutateur d'éclairage (E1) informe le J519. Le J519 pilote ensuite les feux en filaire ou via CAN pour les modules distants.

- Modes de fonctionnement :

  • Normal : Allumage immédiat des feux dès réception du message CAN valide.
  • Défaut : En cas d'interruption de ligne ou court-circuit (ex : Défaut F2 pour clignotant gauche), un code d'erreur est mémorisé dans le J519.
  • Correction : Effacer les codes d'erreur dans les unités 09 (Centrale électrique) et 46 (Confort) après réparation physique.

2. Moteur essuie-glace :

- Fonction :

Nettoyage du pare-brise selon plusieurs vitesses et intermittence.

- Messages CAN :

Les commandes proviennent du J527 (commodo d'essuie-glace) vers le J519 via le bus CAN Confort.

- Interaction ECU :

Le J527 lit la position du levier et envoie l'information au J519, qui pilote le moteur d'essuie-glace.

- Modes de fonctionnement :

  • Normal : Balayage synchronisé avec la commande.
  • Défaut : Défaut F4 (interruption de la ligne de masse). Le système peut se mettre en mode dégradé (vitesse unique) ou s'arrêter.
  • Correction : Vérification de la continuité de la ligne de masse et du câblage moteur.

3. Interrupteur de marche arrière :

- Fonction :

Activer les feux de recul et informer les systèmes d'aide au stationnement.

- Interaction ECU :

Le signal de l'interrupteur est généralement lu par le calculateur de boite ou le J519, puis diffusé sur le bus CAN.

- Modes de fonctionnement :

  • Normal : Allumage des feux de recul dès l'engagement de la marche arrière.
  • Défaut : Défaut F6 (interruption de la ligne positive vers la prise remorque/feux).
  • Correction : Contrôle du fusible et de l'interrupteur sur la boîte de vitesses.

4. Contact / Neiman (E415) :

- Fonction :

Identification de la clé et autorisation de démarrage.

- Interaction ECU :

Le module E415 posséde des microcontacteurs (contact S, clé enfoncée) lus par le Calculateur central de confort (J393).

- Interaction Bus :

Le J393 communique avec le J533 (Passerelle) pour transmettre l'état des bornes (15, 50, X) aux autres calculateurs via les bus CAN Propulsion et Confort.

- Modes de fonctionnement :

  • Normal : Déverrouillage électrique de la colonne de direction (J764) après lecture de la puce de la clé via une bobine LIN.
  • Défaut : Bobine de lecture (01128) défectueuse ou défaut de communication avec le J393.
  • Correction : Diagnostic via le bloc de valeurs mesurées et vérification du bus LIN entre E415 et J393.

5. Feux de détresse :

- Fonction :

Signalisation d'urgence par clignotement simultané de tous les indicateurs de direction.

- Interaction ECU :

L'interrupteur de feux de détresse est relié directement au J519. Le J519 commande ensuite les clignotants et envoie un message sur le bus CAN pour confirmer l'état au combiné d'instruments (J285).

- Modes de fonctionnement :

  • Normal : Clignotement prioritaire sur les autres fonctions d'éclairage.
  • Défaut : Si le bus CAN Confort est défaillant, les feux de détresse peuvent rester opérationnels en mode secours local via le J519.
  • Correction : Remplacement de l'interrupteur ou vérification du câblage vers le J519.
